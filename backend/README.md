#forecast-prediction

Instalação

    Clone o repositório:

    git clone https://github.com/GuilhermeAlencarM/forecast-prediction



Crie e ative o ambiente virtual:

bash

    python3 -m venv venv
    source venv/bin/activate  # Linux/Mac


Instale as dependências:

bash

    pip install -r requirements.txt

Uso

    Execute o programa:

    uvicorn main:app
